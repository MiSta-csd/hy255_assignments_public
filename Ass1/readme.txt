HY255 Εργαστήριο Λογισμικού
Άσκηση 1

Μίνως Σταυρακάκης
csd4120

Η άσκηση υλοποιήθηκε σε πλήρη αρμονία
με τις οδηγίες της εκφώνησης και τις συμβουλές 
που δόθηκαν από το διδάσκοντα κατά την ώρα 
του μαθήματος.

Η αρχική έκδοση του κώδικα της άσκησης χρησιμοποιούσε 
μία συνάρτηση "state FSM (state st, int c)", (όπου state = 
enum state { def, M, N, m, n }), για την υλοποίηση της FSM, 
κάτι το οποίο παρ΄ ότι λειτουργικό αντικαταστάθηκε από 
μία δομή array από pointers σε functions το οποίο εκτός από 
πιο σύντομο, ζητείται και για την απόκτηση του Bonus βαθμού 
της Άσκησης.
